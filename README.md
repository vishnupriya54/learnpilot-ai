# 🚀 LearnPilot AI — Course Recommendation Agent

A personalized AI learning mentor that generates custom learning roadmaps using **Google Gemini AI**.

## ✨ Features

- 🎯 Personalized course roadmaps based on your background & career goal
- 🤖 Powered by Google Gemini 2.5 Flash
- 📚 Smart course ordering by difficulty
- 📄 Download your roadmap as a text file

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini AI (`google-generativeai`)
- python-dotenv

## 🚀 Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/vishnupriya54/learnpilot-ai.git
   cd learnpilot-ai
   ```

2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Create a `.env` file and add your Gemini API key
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

4. Run the app
   ```bash
   streamlit run app.py
   ```
